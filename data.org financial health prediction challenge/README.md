# Data.org Financial Health Prediction Challenge

## Overview
This project contains data and notebook assets for the Data.org Financial Health Prediction Challenge.

## Folder Structure
- `data/` - training and test CSV files
- `notebook/` - experimentation notebook(s)
- `submission/` - generated submission files

## Quick Start
1. Open the notebook in `notebook/`.
2. Train/evaluate your model using `data/Train.csv`.
3. Generate predictions for `data/Test.csv`.
4. Save outputs in `submission/`.
