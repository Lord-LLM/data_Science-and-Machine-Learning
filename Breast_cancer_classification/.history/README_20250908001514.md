# 🩺 Breast Cancer Classification (ML Models)

This project implements **machine learning classification models** to predict whether a tumor is **Malignant (M)** or **Benign (B)** using the [Breast Cancer Wisconsin Dataset](https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data).  

The goal is to build and evaluate different classifiers to assist in **early detection of breast cancer**.

---

## 📂 Dataset
- **Rows:** 569  
- **Features:** 30 (mean, standard error, and worst values of cell nuclei features)  
- **Target:** Diagnosis (`M` = Malignant, `B` = Benign)  

---

## ⚙️ Preprocessing
- Dropped irrelevant columns (`id`, `Unnamed: 32`).  
- Encoded categorical target variable (`M = 1`, `B = 0`).  
- Scaled numerical features with **StandardScaler**.  
- Split dataset into **train/test** (80/20).  

---

## 🧠 Models Used
1. **Logistic Regression**  
2. **Random Forest Classifier**

---

## 📊 Model Performance

### Logistic Regression
- **Accuracy:** `93.86%`
- **Classification Report:**
  - Precision (B): 0.92 | Recall (B): 0.99 | F1-score (B): 0.95  
  - Precision (M): 0.97 | Recall (M): 0.86 | F1-score (M): 0.91  


---
