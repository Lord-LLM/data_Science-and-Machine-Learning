# RHEA Soil Nutrient Prediction

## Overview
This project holds data and notebook files for the RHEA soil nutrient prediction task.

## Folder Structure
- `data/` - train/test and supporting CSV files
- `notebook/` - model development notebook
- `submisssion/` - prediction submission outputs

## Quick Start
1. Inspect metadata in `data/data_dictionary.csv`.
2. Train models using `data/Train.csv`.
3. Predict on `data/TestSet.csv`.
4. Write outputs to `submisssion/`.
