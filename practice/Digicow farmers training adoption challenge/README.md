# Digicow Farmers Training Adoption Challenge

## Overview
This project contains datasets, notebooks, and submission outputs for the Digicow Farmers Training Adoption Challenge.

## Folder Structure
- `data/` - challenge datasets and dictionary
- `notebooks/` - solution notebook(s)
- `submision/` - submission artifacts

## Quick Start
1. Explore schema in `data/dataset_data_dictionary.csv`.
2. Build and validate features from `data/Train.csv`.
3. Predict on `data/Test.csv`.
4. Export submission files to `submision/`.
